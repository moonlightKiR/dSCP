\section{El Problema}

El objetivo es \textbf{diseñar una herramienta de análisis de datos} que, mediante el procesamiento de una fotografía facial, sea capaz de clasificar a un individuo y devolver un porcentaje de \textit{"probabilidad de tendencia criminal"}.

El sistema actuará como un recomendador que asocia \textbf{rasgos biométricos} específicos con etiquetas de conducta extraídas de bases de datos históricas.

El dominio del problema es a \textit{Biometría Forense} aplicada a la \textbf{Seguridad} y la \textbf{Psicología de la Percepción}, buscamos extraer patrones faciales para resolver un problema de \textbf{identificación de riesgos}.
