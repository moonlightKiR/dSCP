\section{Abstract}

Este estudio utiliza redes neuronales convolucionales (\textbf{ResNet50}) para clasificación biométrica, eliminando el sesgo de entorno (\textit{shortcut learning}). Mediante MTCNN se aislaron los rostros, forzando a la red a procesar rasgos faciales y no fondos institucionales. Así, el aprendizaje se basa exclusivamente en la morfología del sujeto.

Se aplicó un balanceo demográfico estricto para mitigar sesgos raciales en la red neuronal. Mediante auditoría étnica, se conformó un dataset equitativo que neutraliza prejuicios estadísticos. Esto garantiza que las neuronas del modelo respondan a patrones universales, promoviendo una clasificación justa y técnicamente robusta.

El modelo alcanzó un 97\% de precisión tras el balanceo. La validación con Grad-CAM confirmó que la red focaliza su atención en zonas clave como la estructura ósea y los ojos. Se concluye que las redes neuronales requieren equilibrio poblacional y limpieza visual para evitar decisiones basadas en sesgos.
