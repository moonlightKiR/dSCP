\section{Experimentación / Resultados obtenidos}

\subsection{Diseño experimental}

El rigor científico del estudio se sustenta en una metodología de validación diseñada para evaluar la capacidad de generalización de los modelos y la ausencia de sesgos discriminatorios.

\subsubsection{Metodología de Entrenamiento y Test}

Se ha implementado un método de validación \textbf{Hold-out 80/20}. El dataset balanceado se divide aleatoriamente en un $80\%$ para el entrenamiento y un $20\%$ para el test. Esta partición garantiza que el modelo sea evaluado con rostros que no han intervenido en el proceso de aprendizaje. El entrenamiento se ejecuta durante un ciclo de épocas controlado con un tamaño de lote (\textit{batch size}) de 64 imágenes para estabilizar el gradiente.

\subsubsection{Métricas de Evaluación}

Para una auditoría exhaustiva del rendimiento, el sistema genera automáticamente un informe analizando la \textbf{Exactitud (Accuracy)}, el \textbf{F1-score} (media armónica entre precisión y exhaustividad) y la \textbf{Matriz de Confusión}, representación visual clave para identificar si el modelo tiene sesgos hacia alguna de las clases.

\subsection{Resultados y análisis}

Tras la ejecución de los experimentos sobre las arquitecturas ResNet50 y VGG-16, se han extraído datos que confirman la viabilidad de la propuesta técnica.
