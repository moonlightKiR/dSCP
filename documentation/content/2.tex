\section{La Aplicacón}

La aplicación que se abordará en este proyecto es la \textit{Clasificación Binaria} mediante aprendizaje supervisado.

El sistema procesará imágenes faciales etiquetadas previamente en dos categorías \textbf{Perfil de Riesgo} y \textbf{Perfil Estándar} para entrenar un modelo capaz de asignar una etiqueta de probabilidad a nuevos sujetos no vistos anteriormente.
